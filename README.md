# OfflineSyncKitDemo

A minimal notes app built to demonstrate OfflineSyncKit in action.

## What it shows
Create and delete notes that persist locally even without internet. Sync status updates in real time at the top of the screen. When network returns, pending operations sync automatically.

## How to run
Clone this repo and OfflineSyncKit into the same folder.  
Open OfflineSyncKitDemo.xcodeproj in Xcode.  
Build and run on an iOS 17 simulator or device.  

OfflineSyncKit repo: https://github.com/sushanttiwari-14/OfflineSyncKit

## Requirements
iOS 17+  
Xcode 15+
